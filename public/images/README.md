# Site images

Drop your photos in **this folder** using the exact filenames below. They are
served from `/images/<name>` and replace the placeholders automatically — no code
change needed. Until a file exists, the site shows an on‑brand fabric placeholder
(never a broken image).

**Format:** `.jpg` (or `.webp` — keep the same base name). Keep each file under
~300 KB for fast loads. Suggested sizes are guides, not hard limits.
